Клеточный автомат с настраиваемыми правилами и выбором цветов
Проект представляет собой интерактивную симуляцию многодветного клеточного автомата, вдохновлённую генерацией биомов. Пользователь может в реальном времени настраивать правила перехода между двумя выбранными цветами, изменять их вероятности, а также менять отображаемую цветовую пару.

https://screenshot.png

Возможности
Два активных цвета – пользователь выбирает пару цветов (A и B), которые взаимодействуют между собой. Все остальные цвета игнорируются в симуляции, но могут присутствовать на поле (при случайном заполнении или рисовании).

Настройка правил перехода – для каждого направления (A → B и B → A) задаются:

Список количеств соседей того же цвета, при котором срабатывает правило.

Вероятность перехода в другой цвет (от 0 до 1).

Графический интерфейс настройки – модальное окно с ползунками и текстовыми полями (библиотека raygui).

Управление симуляцией:

Play/Pause (пробел или кнопка)

Step (один шаг, клавиша N или кнопка)

Clear (очистить поле, клавиша C)

Random (заполнить поле случайными цветами A/B с вероятностью 0.5, клавиша R)

Редактирование поля мышью:

Левая кнопка – поставить цвет A

Правая кнопка – поставить цвет B

Навигация:

Колесо мыши – масштабирование

Стрелки – панорамирование

Выбор отображаемых цветов (кнопки A: и B:) – меняются только отображаемые цвета, правила перестраиваются для новой пары.

Технологии
Язык: C++17

Графика: Raylib 6.0

GUI: raygui (header‑only)

Компилятор: MinGW-w64 (ucrt64) или GCC

Сборка и запуск
Требования
Установленный компилятор с поддержкой C++17.

Библиотека Raylib (скомпилированная для вашей системы).

Инструкция для Windows (MSYS2)
Установите MSYS2 и необходимые пакеты (см. официальную документацию Raylib).

Скачайте и распакуйте Raylib (например, в C:/raylib/raylib-6.0_win64_mingw-w64).

Склонируйте репозиторий проекта.

Перейдите в папку проекта в терминале MSYS2 UCRT64.

Выполните компиляцию:

bash
g++ -g main.cpp Cell/Cell.cpp Rules/Rules.cpp GameGrid/GameGrid.cpp GameRenderer/GameRenderer.cpp Game/Game.cpp raygui_impl.cpp -o game_of_life.exe -I/c/raylib/raylib-6.0_win64_mingw-w64/include -L/c/raylib/raylib-6.0_win64_mingw-w64/lib -lraylib -lopengl32 -lgdi32 -lwinmm
Убедитесь, что raylib.dll находится в той же папке, что и исполняемый файл (или в PATH).

Запустите:

bash
./game_of_life.exe
Примечание о raygui.h
Файл raygui.h должен находиться в папке Game/. Исходный код уже включает его (см. Game/raygui.h). Файл raygui_impl.cpp определяет RAYGUI_IMPLEMENTATION и подключает raygui.h.

Управление
Действие	Клавиша / Мышь
Play / Pause	Пробел (или кнопка Play/Pause)
Шаг симуляции	N (или кнопка Step)
Очистить поле	C (или кнопка Clear)
Случайное заполнение	R (или кнопка Random)
Открыть окно правил	S (или кнопка Rules)
Увеличить / уменьшить	Колёсико мыши
Панорамирование	Стрелки
Поставить цвет A	Левая кнопка мыши
Поставить цвет B	Правая кнопка мыши
Сменить цвет A	Кнопка "A:" (левая кнопка)
Сменить цвет B	Кнопка "B:" (левая кнопка)
Выход	ESC
Настройка правил
При нажатии кнопки Rules (или клавиши S) открывается модальное окно. В нём можно:

Выбрать редактируемый цвет (переключатели < и >).

Задать список количеств соседей (через пробел).

Изменить вероятность перехода (ползунок).

Указать целевой цвет (кнопки < и >).

Сохранить изменения или отменить.

После сохранения правила применяются к текущей паре активных цветов (A и B). При смене A или B правила автоматически перестраиваются для новой пары (с сохранёнными для этих цветов параметрами).
